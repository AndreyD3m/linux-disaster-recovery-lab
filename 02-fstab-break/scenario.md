# Сценарий: Сломанный fstab

## Цель
Вывести систему из строя, добавив нерабочую запись в `/etc/fstab`, и восстановить доступ через emergency mode.

## Окружение
- Виртуальная машина: VirtualBox 7.1
- ОС: Ubuntu 20.04 (focal)
- RAM: 2 ГБ
- Диск: 20 ГБ

## Шаги воспроизведения (поломка)

1. Открыть файл fstab:
   ```bash
   sudo nano /etc/fstab
2. Добавить в конец файла нерабочую строку:
    ```text
    /dev/sdX99  /mnt/test  ext4  defaults  0 0
3. Сохранить файл и перезагрузить систему:
    ```bash
    sudo reboot
