# Решение: Восстановление после сломанного fstab

## Если система загрузилась в emergency mode

1. Ввести пароль root (или пользователя) для получения shell.

2. Открыть файл fstab:
   ```bash
   nano /etc/fstab
3. Удалить или закомментировать (# в начале) нерабочую строку.
4. Сохранить файл и перезагрузить:
    ```bash
    reboot
1.  Если система не загружается даже в emergency mode
# Загрузиться с LiveCD Ubuntu.
2. Примонтировать корневой раздел:
    ```bash
    sudo mount /dev/sda2 /mnt
3. Отредактировать fstab внутри примонтированной системы:
    ```bash
    sudo nano /mnt/etc/fstab
4. Удалить мусорную строку, сохранить.
5. Перезагрузиться:
    ```bash
    sudo umount /mnt
    sudo reboot
