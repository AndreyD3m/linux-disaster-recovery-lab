# linux-disaster-recovery-lab
Репозиторий с лабораторными работами по системному администрированию Linux. Содержит сценарии для воспроизведения критических сбоев и подробные инструкции по восстановлению. Цель — отработка навыков диагностики и ремонта Linux-систем в условиях, приближенных к реальным.
