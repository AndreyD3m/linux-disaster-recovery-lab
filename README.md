# linux-disaster-recovery-lab
Репозиторий с лабораторными работами по системному администрированию Linux. Содержит сценарии для воспроизведения критических сбоев и подробные инструкции по восстановлению. 
# Цели проекта
* Диагностировать критические сбои Linux-систем
* Восстанавливать работоспособность после фатальных ошибок
* Понимать внутренние механизмы ядра, загрузчика, менеджера пакетов и памяти
* Чувствовать себя уверенно перед лицом самой страшной ошибки
# Содержание
1. Тема: Удаление всех ядер  Описание: Полное удаление ядер Linux, загрузка в Memtest86+, восстановление через LiveCD
2. Тема: Сломанный fstab  Описание: Некорректная запись в /etc/fstab, попадание в emergency mode, починка
3. Тема: Перегрузка оперативной памяти  Описание: Создание процесса, который нагружает оперативную память с помощью stress-ng и принудительное завершение процесса.
4. Тема: Заполнение диска  Описание: Заполнение диска до отказа, поиск "зомби-файлов" через lsof
