    df -h

2. Найти самые большие файлы и папки
    ```bash
    du -ah ~/ | sort -rh | head -n 10 # Топ-10 самых тяжёлых файлов в домашней папке
3. Способ 1. Просто удалить большой файл
    ```bash
    rm -f ~/bigfile

4. Поиск открытых удалённых файлов (если место не освобождается после удаления)
    ```bash
    lsof | grep deleted
Эта команда покажет файлы, которые удалены, но всё ещё заняты процессами
5. Перезапустить процесс или убить его:
    ```bash
